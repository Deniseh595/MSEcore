Materials Science and Engineering Core Curriculum

Editor: Kenneth R. Shull, Northwestern University
This page includes links to the course content for the core required classes in Materials Science and Engineering at Northwestern University. The curriculum summary documents include aggregated information from all of the classes, and the individual course documents include documents for the individual courses. Some of the links are to pdf files that can be viewed directly in your browser, or saved by right clicking on the link and choosing the 'save link as' option.

Curriculum Summary Documents

These documents include aggregated content from the entire core course curriculum:
  • Laboratories (pdf): This is a useful document if you want to see the complete laboratory curriculum all in one place.
  • Problems (pdf): An aggregated list of the problems assigned throughout the curriculum. It can be used as a practical guide to the problems that you will be expected to be able to solve after completing the curriculum.
  • Catalog Descriptions: Course descriptions as they appear in the university catalog.
  • Course outcomes: Course outcomes for each of the core classes. These outcomes describe things you should be able to do after completing the core curriculum.
  • Complete MSE course text (pdf): All of the individual course texts combined into a single pdf file. It's big ( ∼ 1200 pages and ∼ 80 MBytes as of fall, 2015), and is currently under a state of active development. Nevertheless, it provides a useful single source describing the intellectual content of our discipline, with a table of contents, list of symbols, index, etc. It is expected that current students will primarily access individual course documents from the links given below.

Combined Course Documents

In some cases it is useful to combine texts for related courses into a single pdf file. We may come up with more of these, but for now we just include one that includes a combined text for the 314, 315, 316-1,2 sequence.
  • Combined text for 314, 315, 316-1,2 This document combines the course text documents for 314, 315 and 316-1,2 into a single document, which is being developed as a standalone text to provide a coherent framework across the core course sequence that deals with thermodynamics, kinetics, and defect structure in materials.

Computational Materials Science

  • matlab tutorial and examples (pdf): Some tutorial information, with some MATLAB example code relevant to the MSE core curriculum.
  • assigned problems for computational MSE (pdf): Assigned problems relating to computational MSE.
